TEST 3: POST /stream (SSE streaming)
--------------------------------------------------------------
$ curl -s -N -X POST http://localhost:8000/stream \
  -H "Content-Type: application/json" \
  -d '{"message": "What is the SOP for a logistics delay?", "mode": "single"}'

data: {"type": "tool_call", "node": "agent", "tool": "search_sop_wiki", "args": {"query": "logistics delay protocol"}}

data: {"type": "tool_result", "node": "tools", "tool": "search_sop_wiki", "content": "SOP-002: Logistics Delay Response Protocol\n1. Confirm delay duration..."}

data: {"type": "agent_response", "node": "agent", "content": "Here is the Standard Operating Procedure for handling a logistics delay:\n\n**SOP-002: Logistics Delay Response Protocol**\n1. Confirm delay duration and root cause..."}

data: {"type": "done"}

Status: 200 OK (text/event-stream)

SUMMARY

All 3 API tests passed successfully.
